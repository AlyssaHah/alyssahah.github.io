'Check That Stock!' Mockup
Previous Next

Here are some pictures of a mockup app I created using Adobe XD.

This is an app that people could use to see which stores have what items currently in stock. This would allow people to save time in running to the store without knowing what each store currently has for sale. The app will automatically take and calculate the values of each individual store's current inventory, using the technology that stores already implement. The app would sync up to the information, and give each employee of the store the ability to monitor and manually update each total. The user can choose which specific store they wish to look at using Google Maps, and upon selecting the pin icons will be brought to each store's specific stock page.
